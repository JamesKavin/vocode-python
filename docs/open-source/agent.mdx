## Overview

Agents represent an abstraction for an AI (usually an LLM) that powers the content
generation/understanding layer of the conversation. In other words, Agents are the
AI that is actually having the conversation.

Vocode provides agents that work out-of-the-box (see below) and also allows developers to
[create their own agents](/create-your-own-agent).

## Default Agent Implementations

All Agents are implemented using the corresponding `AgentConfig` class.

### ChatGPTAgentConfig

The `ChatGPTAgentConfig` is configured to use the new ChatGPT endpoint from OpenAI.

### LLMAgentConfig

The `LLMAgentConfig` is configured to use the completions endpoint using the simpler `text-curie-001` model.
