We provide [playground scripts](https://github.com/vocodedev/vocode-python/tree/main/playground) to individually
test transcribers, agents, and synthesizers.

To begin, clone the [repo](https://github.com/vocodedev/vocode-python).

# Streaming

## Transcriber

Run the following script to transcribe your microphone input:

```
python playground/streaming/transcriber/speak.py
```

## Agent

Run the following script to interact with your agent in text format:

```
python playground/streaming/agent/chat.py
```

## Synthesizer

Run the following script to synthesize text to speech and play it to your speaker:

```
python playground/streaming/synthesizer/listen.py
```
